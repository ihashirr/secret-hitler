# ECLIPSE: Mobile-First Design Strategy

## 1. Ergonomic Architecture (The 20/35/45 Rule)
- **Top (20%)**: Header / Branding / Passive Status.
- **Middle (35%)**: Atmospheric Feedback / Scrolling Logs / Cinematic Depth.
- **Bottom (45%)**: **The Thumb-Zone**. All interactive elements (inputs, buttons) are clustered here for one-handed native-app quality feel.

## 2. The "Entry Ritual"
- **Sequential Infiltration**: Replaces static forms with a multi-step "handshake" (Identification → Frequency Sync → Neural Uplink).
- **Tactical Feedback**: Every user input triggers a visual reaction in the Atmospheric Hub.
- **Reduced Cognitive Load**: One task at a time, moving the interface to the thumb, not the thumb to the interface.

## 3. Visual Identity & Depth
- **Tactical HUD Aesthetic**: Deep obsidian bases with high-contrast cyan neon accents.
- **Cinematic Layering**: Using scanlines, glitch flickers, and perspective grids to create the feel of a high-end surveillance terminal.

## 4. Linguistic Immersion (Tactical Copy)
**Replace generic web-form language with system-driven narrative:**
- `Join/Connect` → `INITIALIZE_LINK`
- `Room Code` → `FREQUENCY_BAND`
- `Name` → `IDENTIFY_OPERATIVE`
- `Error` → `HANDSHAKE_REFUSED`

## 5. Mobile-Specific Polish
- **100dvh Compliance**: Ensures the UI is always correctly sized for mobile browser chrome.
- **Magnetic Tap Targets**: Large, high-precision activation zones for buttons.
- **High-Speed Staggers**: Intentional, precise, and system-driven animations.
